La adición de imágenes.

Además de vincular a otras páginas, támbien podemos usar URL para agregar imágenes a una página web.

Para agregar imágenes a una página web comenzamos con la etiqueta img.

Al igual que "br", "img" es una etiqueta vacía.

Eso Significa que no tiene etiqueta de cierre

Para mostrar una imagen, una etiqueta de imagen necesita el atributo src, el cuál significa fuente.



Establecemos la imagen que queremos mostrar con un signo = y la dirección entre comillas dobres "".

Drawing of Earthrise



Podemos utilizar atributos para cambiar el tamaño de las imágenes. Los atributos width(ancho) y height(altura), el cuál usan pixeles como unidad de medida predeterminada.

Por ejemplo, para establecer el ancho de esta imagen en 100 px de ancho, lo podemos aumentar a 200px.



//SEGUNDA LECCIÓN DE IMAGÉNES

Las URL como https://mimo.app/r/earth.png acceden acceden a imágenes en computadoras que pueden estar a miles de kilómetros de distancia.

https://mimo.app/ apunta a la computadora, r/ es una carpeta almacenada en la computadora y earth.png es el nombre del archivo.


Del ejemplo anterior https://mimo.app/r/earth.png, para acceder a archivos locales, o archivos que existen en nuestros dispositivos, omitimos https://mimo.app y solo escribimos /r/earth.png.


//REPASO

Los atributos añaden información adicional a las etiquetas.
